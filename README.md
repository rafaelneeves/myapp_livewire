## To-do List Livewire

Criei este to-do list com o intuito de conhecer o framework livewire. <br>
O To-do List Livewire é um crud que insere, atualiza, lista e deleta um lista de coisas. 

## Requisitos

- Laravel >= 9.19
- Php >= 8.0.2
- Livewire >= 2.10
- Composer

## Rodar Projeto
- Clone o Repositório
- Baixe as dependências: composer install
